STM32WBA Web Bluetooth® App Interfaces

A live STMicroelectronics demonstration

This demonsration combines three ST technologies to showcase real-time wireless performance. It measures, processes, and displays results live in the browser.

Objective

  • Instant short-range data transfer with ST60 (60 GHz)
  • Edge processing and measurement on STM32MP2
  • Live visualization via BLE with Nucleo-WBA55CG

Designed for a showroom: simple to run, visually compelling, and technically solid.

Why this demonsration stands out

  • Cable-free at 60 GHz: ST60 replaces short wired links with multi-Gb/s wireless.
  • Low latency (sub-ms typical): actions and results feel immediate.
  • Web BLE interface: scan a QR code and view live metrics on compatible devices.
  • Industry 4.0 / IoT ready: a convincing path to smarter, faster systems.

Technology overview — one cohesive system

ST60 — short-range 60 GHz link
Data rate up to ~6 Gb/s (application-dependent) • Sub-ms latency (typical) • Low power
Role: carries Ethernet data to the STM32MP25 for measurement.

STM32MP2 — the processing hub
Cortex-A35 + Cortex-M33 • Linux • PCIe, USB 3.0, Ethernet
Role: runs ping and iperf3, aggregates throughput & latency, streams results over UART.

Nucleo-WBA55CG — BLE made simple
BLE 5.3 • low power • custom GATT
Role: publishes throughput (D) and latency (L) to the browser via Web Bluetooth®.

A visual interface anyone can use

  • Web Bluetooth® UI via QR code
  • Live charts (React + Chart.js), zero install
  • Browser support: Chrome / Edge (desktop & Android). iOS Safari not officially supported.

Showroom experience

  1. Scan the QR code and connect to the demo board over BLE.
  2. Watch latency update every second.
  3. Tap “Measure throughput (10 s)” to trigger iperf3 and see live Mb/s.
  4. If the link is interrupted, charts automatically fall back to 0 until recovery.
